# AI-консультант для ГК Проект
## План перехода MVP → Продакшен

---

### О продукте

AI-консультант — интеллектуальный чат-виджет, который устанавливается на сайт gkproject.ru и автоматически консультирует посетителей по услугам компании: монтаж котельных, отопление, водоснабжение, электрика, канализация.

**Возможности:**
- Ответы на вопросы клиентов 24/7 на основе базы знаний компании
- Голосовой ввод и озвучка ответов
- Сбор заявок (имя, телефон) с мгновенной отправкой в Telegram-группу отдела продаж
- Быстрые кнопки: «Какие услуги?», «Сколько стоит монтаж?», «Гарантии» и др.
- Встраивание на сайт одной строкой кода (iframe)

**Демо:** https://ai.pavelkarikoff.ru

---

### 1. Инфраструктура

| Параметр | MVP (текущий) | Продакшен |
|----------|---------------|-----------|
| Сервер | 1 VPS, все сервисы в Docker | 1–2 VPS (фронтенд + бэкенд), возможность масштабирования |
| Резервирование | Нет | Ежедневные бэкапы БД, логов и конфигураций |
| SSL-сертификат | Let's Encrypt (бесплатный) | Let's Encrypt на домене клиента (chat.gkproject.ru) |
| Оперативная память | ~400 МБ | 2–4 ГБ с мониторингом нагрузки |

---

### 2. Подключение к сайту gkproject.ru

Для установки виджета на сайт достаточно добавить одну строку в HTML-код (совместимо с Битрикс и любой CMS):

```html
<iframe src="https://chat.gkproject.ru/?embed=1"
  style="position:fixed;bottom:0;right:0;width:400px;height:560px;border:none;z-index:9999">
</iframe>
```

Для корректной работы клиент создаёт поддомен `chat.gkproject.ru` с A-записью на IP продакшен-сервера. SSL-сертификат выпускается автоматически и бесплатно.

**Срок подключения:** 1 рабочий день.

---

### 3. База данных (доработка для продакшена)

| Компонент | MVP | Продакшен |
|-----------|-----|-----------|
| Сессии чата | В оперативной памяти (теряются при перезапуске) | PostgreSQL / Redis |
| Заявки (лиды) | В оперативной памяти | PostgreSQL с историей и аналитикой |
| База знаний (RAG) | ChromaDB на файловой системе | ChromaDB на отдельном томе с бэкапами |
| Логи разговоров | JSONL-файлы | Централизованное хранилище логов |

---

### 4. Безопасность

- Авторизация API через JWT-токены (вместо простого ключа)
- Rate limiting — защита от спама и перегрузки
- CORS — ограничение доступа только с домена gkproject.ru
- Экранирование пользовательского ввода (защита от XSS и инъекций)
- Соответствие 152-ФЗ (обработка персональных данных)
- Согласие на обработку ПД в форме заявки

---

### 5. Мониторинг и поддержка

- **Доступность:** автоматическая проверка каждые 30 секунд, алерт в Telegram при сбое
- **Метрики:** время ответа, количество запросов, процент ошибок (Grafana)
- **Расходы AI:** отслеживание потребления токенов OpenAI, бюджетные лимиты
- **Обновление базы знаний:** автоматический парсинг gkproject.ru при изменении контента

---

### 6. Интеграции

| Интеграция | Описание |
|------------|----------|
| **Telegram** | Мгновенная отправка заявок в группу отдела продаж (уже работает) |
| **CRM (Битрикс24 / amoCRM)** | Автоматическое создание лидов и сделок |
| **Google Sheets** | Дублирование заявок в таблицу (уже работает) |
| **Яндекс.Метрика** | Отслеживание конверсий из виджета |
| **Голосовой ввод** | Распознавание речи (OpenAI Whisper) и озвучка ответов (OpenAI TTS) |

---

### 7. Масштабирование

При росте нагрузки (более 100 одновременных пользователей):
- Несколько экземпляров бэкенда за балансировщиком нагрузки
- Кэширование частых запросов (Redis)
- CDN для статических файлов (Cloudflare, бесплатно)
- Очередь задач для тяжёлых операций (распознавание речи, синтез)
